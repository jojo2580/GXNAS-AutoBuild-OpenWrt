## `OpenWrt-GXNAS-6.0内核`

- 默认IP地址：`192.168.1.1`
- 账户：`root`
- 密码：`空`
 
 ================================================================
- 首先需要打开 Openwrt 主页,点击系统-TTYD 命令窗,或者使用putty和openwrt后台luci插件定时更新 
- 输入`openwrt`即可进入固件升级菜单                            
- 输入`tools`即可打开工具箱
- 输入`qinglong`即可全自动安装青龙 
- ================================================================
